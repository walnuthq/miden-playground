Now that our custom note script is compiled, let's create the note by executing a custom transaction.
You will need a connected wallet to sign the transaction so make sure your Miden Wallet is connected to the playground before continuing.

From the <Link href="/notes">notes page</Link>, click on the _"Add note"_ button and select the _"Create note"_ option.

A _"Create Note"_ dialog will show up to help you configure the note creation transaction.
Set the recipient account to **Test Wallet** and select the timelock script, leave the note type as public.

The note will be created through your wallet so you must include some of your assets in the note, try sending **10 MIDEN** tokens.

The final step is to add the correct note storage to configure your custom note.

The first two storage items are the target account ID suffix and prefix, you can use the <Link href={`/accounts/${props.address}`}>Test Wallet</Link> account ID suffix **{props.accountIdSuffix}** and prefix **{props.accountIdPrefix}**.

The third storage item is the timelock block height, you can set it to any future block height eg. **{props.syncHeight}**.

Once you're done configuring your note creation, click on the _"Create"_ button and sign the resulting transaction.
