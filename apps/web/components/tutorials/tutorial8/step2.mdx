From the <Link href="/accounts">accounts page</Link>, click on _"Create new account"_ and select the _"Deploy account"_ option.

Give your new contract a name and select **Network** as storage mode.
Choose **NoAuth** for the authentication scheme and select the custom **Counter Contract** account component.

When you're done configuring your contract deployment, click on the _"Deploy"_ button to create your network account.
The key difference from regular contracts is using the **Network** [Account Storage Mode](https://docs.miden.xyz/miden-base/account/id#account-storage-mode) instead of **Public**, which enables the contract to be executed by the network operator.
