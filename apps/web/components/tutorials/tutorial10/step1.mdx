In this tutorial, you'll create your first smart contract and understand the structure and implementation of both the counter account contract and increment note script.

Let's get started by clicking on _"Create new script"_ then make sure to name your script **counter-account** and select an **Account Component** script with the **Counter Contract** example.

You must name your script **counter-account** as we'll create other resources (storage slot names, depending scripts, etc.) that expect this particular name.
